## Сообщение об уязвимостях

Если вы обнаружили уязвимость безопасности, пожалуйста, **не создавайте публичный issue**. Вместо этого:

1. Отправьте описание проблемы мейнтейнерам по email
2. Предоставьте подробное описание уязвимости
3. Включите шаги для воспроизведения (если применимо)
4. Дайте 90 дней на исправление до публичного раскрытия

## Лучшие практики для контрибьюторов

### Предотвращение утечки учётных данных

Этот репозиторий использует SSH-аутентификацию.

#### ✅ Рекомендуется: SSH

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub
git clone git@github.com:nagual2/font_sharpener.git
```

#### ❌ Не рекомендуется: HTTPS с токенами

Никогда не встраивайте токены в URL:
```bash
# НИКОГДА ТАК НЕ ДЕЛАЙТЕ
git clone https://username:TOKEN@github.com/user/repo.git
```

### Сканирование секретов

```bash
git diff --cached | grep -iE "password|token|api.?key|secret|credential"
gitleaks detect --staged
```

### Чеклист ревью кода

- [ ] Нет паролей или токенов
- [ ] Нет API-ключей в коде
- [ ] Нет приватных ключей
- [ ] Нет персональных данных
- [ ] Комментарии не содержат чувствительной информации
